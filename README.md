# tutorial_bubbles

`tutorial_bubbles` is a Flutter package for building guided tutorial bubbles that highlight widgets in your app.

It provides a configurable bubble, arrow, and overlay system that can be used both as a standalone spotlight and as part of a multi-step tutorial engine.

## Installation

Add this to your `pubspec.yaml` in the `dependencies` section:

```yaml
dependencies:
  tutorial_bubbles: ^0.0.1
```

Then run:

```bash
flutter pub get
```

## Usage

See the `example` app in this repository for a minimal demonstration of how to display a tutorial bubble pointing at a target widget.
